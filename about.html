About this project

A focused, practical mini-course that teaches the essentials: structure (HTML), design (CSS), and how they work together.

Mission

Make HTML and CSS feel clear, practical, and approachable. Help learners build confidence through small, interactive wins.

Audience

Students, hobbyists, and first-time web developers who want a gentle, hands-on introduction to building web pages.

What you'll learn

Structure with HTML, styling with CSS, and how to combine them to create simple, responsive pages.

Completed lessons

Course flow

  1. ✅ Lesson 01 - HTML Basics
  2. ✅ Lesson 02 - CSS Basics
  3. ✅ Lesson 03 - HTML + CSS Together

Why this order

Structure first (HTML), then design (CSS), then combine them. This sequence helps beginners build a mental model that scales.

Final update & checklist

✅ Navigation works between all three lessons

✅ Run and Copy buttons work correctly

✅ Mobile responsive layout

✅ Consistent colors, fonts, and spacing

✅ Footer on every lesson

✅ "Next Lesson" on Lessons 1 and 2

✅ Completion message at the end of Lesson 3
