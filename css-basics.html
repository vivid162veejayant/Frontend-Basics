Lesson 02

CSS Basics

Learn how CSS styles HTML — the design layer for the web.

Section 1 - What is CSS?

📖 Explanation

CSS stands for Cascading Style Sheets. CSS controls the appearance of HTML. HTML = structure. CSS = design.

💻 Code Example

h1{ color: blue; }

👀 Output

Example heading

💡 Tip

Think: HTML builds the page, CSS makes it look good.

Section 2 - CSS Syntax

📖 Explanation

Every CSS rule has a selector, and a block of declarations made of property and value.

💻 Code Example

p{ color: red; }

👀 Output

This paragraph will become red when you run the example.

💻 Diagram

p {
  color: red;
}

↑
Selector

      ↑
   Property

             ↑
           Value

💡 Tip

Every declaration ends with a semicolon ;.

Section 3 - Selectors

📖 Explanation

Basic selectors let you target elements: element selectors, class selectors, and ID selectors.

💻 Code Example

h1{ color: green; } .title{ font-size: 30px; } #main{ background: black; color: white; }

👀 Output

Green heading with .title

The #main area has a dark background.

💡 Tip

Use classes for reusable styles and IDs for unique elements on the page.

Section 4 - Colors

📖 Explanation

Use color for text color and background-color for element backgrounds.

💻 Code Example

body{ background-color: lightblue; } h1{ color: darkblue; }

👀 Output

Blue page heading

The page background is light blue.

💡 Tip

Use named colors, hex, rgb, or hsl — whichever you find easiest to read and maintain.

Section 5 - Fonts

📖 Explanation

Control typography with font-size, font-family, and font-weight.

💻 Code Example

p{ font-size: 20px; font-family: Arial, sans-serif; font-weight: bold; }

👀 Output

This is a large bold paragraph.

💡 Tip

Provide fallback fonts (e.g., Arial, sans-serif) so text looks consistent across systems.

Section 6 - Backgrounds

📖 Explanation

Backgrounds can be simple colors or images. We'll keep it basic: background-color and background-image.

💻 Code Example

body{ background-color: #eeeeee; }

👀 Output

Page with a light gray background.

💡 Tip

Keep backgrounds subtle for better readability. Gradients and complex images can come later.

Section 7 - Margin & Padding

📖 Explanation

Margin creates space outside an element. Padding creates space inside an element.

💻 Diagram

+-------------------------+
|       Margin            |
|  +-------------------+  |
|  |     Border        |  |
|  | +---------------+ |  |
|  | |   Padding     | |  |
|  | | +-----------+ | |  |
|  | | | Content   | | |  |
|  | | +-----------+ | |  |
|  | +---------------+ |  |
|  +-------------------+  |
+-------------------------+

💻 Code Example

div{ margin: 20px; padding: 20px; }

👀 Output

Box with margin outside and padding inside.

💡 Tip

Use margin to separate elements from others; use padding to give content breathing room inside its box.

Section 8 - Borders

📖 Explanation

Use border to draw lines around elements and border-radius to round corners.

💻 Code Example

div{ border: 2px solid black; border-radius: 10px; }

👀 Output

Rounded bordered box

💡 Tip

Combine border styles, widths, and radii to create subtle UI elements.

Section 9 - External CSS

📖 Explanation

Keeping CSS in a separate file (styles.css) keeps your project organized and makes styles reusable across pages.

💻 Example files

<!-- index.html --> <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width,initial-scale=1"> <title>External CSS Example</title> <link rel="stylesheet" href="styles.css"> </head> <body> <h1>Hello with external CSS</h1> </body> </html>
/* styles.css */ h1{ color: purple; }

👀 Note

📄 This example uses two separate files (index.html and styles.css).

Copy both examples into separate files to see the result in your browser. There is no single-run preview here because the files must be saved separately.

💡 Tip

External CSS is the real-world workflow — it keeps HTML clean and styles centralized.

Section 10 - Summary

📖 Explanation

Quick review of what you learned about CSS basics.

💻 Key points

  • CSS styles HTML.
  • CSS uses selectors to target elements.
  • Properties change appearance (color, fonts, spacing).
  • External CSS keeps projects organized.
  • Margin creates outside space; padding creates inside space.
  • Borders surround elements and can be rounded.

💡 Tip

Practice by styling small pages. Start with layout and spacing, then refine colors and typography.
