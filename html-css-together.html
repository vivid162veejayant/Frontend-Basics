Lesson 03

HTML + CSS Together

HTML creates the page. CSS makes it readable, usable, and visually clear.

How they work together

HTML and CSS are designed to work as a team. HTML creates the structure and content of a web page. CSS controls the appearance of that content, including colors, fonts, spacing, alignment, and layout.

HTML = what is on the page

CSS = how the page looks

How CSS styles HTML

CSS selects HTML elements and applies visual rules to them. It does not create the content. It changes how existing content appears in the browser.

  • CSS can change the color of headings.
  • CSS can adjust spacing between sections.
  • CSS can center text or images.
  • CSS can add backgrounds, borders, and layout rules.

Before and after styling

HTML only

  • Text uses browser default styling.
  • Spacing is limited.
  • Content is usually left aligned.
  • The page can feel unfinished.

HTML + CSS

  • Headings stand out clearly.
  • Text is easier to read.
  • Sections are visually separated.
  • The design feels more professional.

Why using both matters

  • HTML keeps content structured and meaningful.
  • CSS lets you control design without rewriting content.
  • External styles make multiple pages feel consistent.
  • Good styling improves readability and user experience.
